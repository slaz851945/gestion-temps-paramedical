# 12. Planning prévisionnel

| Phase | Durée estimée |
|-------|---------------|
| Cadrage, conception détaillée et maquettage UI/UX | 3 semaines |
| Mise en place environnement | 1 semaine |
| Développement itératif (6 sprints) | 15 semaines |
| Tests et recette | 3 semaines |
| Formation et déploiement | 2 semaines |
| **Total** | **24 semaines** (environ 6 mois) |

*Note : Ce planning inclut les fonctionnalités avancées (contraintes, workflow, simulation) et le travail de conception UI/UX. Une version MVP (Minimum Viable Product) avec les fonctionnalités essentielles pourrait être livrée en 14-16 semaines.*
