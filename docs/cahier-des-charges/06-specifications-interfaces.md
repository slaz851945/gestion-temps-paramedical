# 6. Spécifications détaillées des interfaces (maquettage)

Des maquettes seront réalisées lors de la phase de conception. À titre indicatif, voici les écrans principaux :

- **Dashboard** : Vue d'ensemble avec alertes (conflits), prochains cours, taux d'occupation, rappels.
- **Calendrier** : Grille hebdomadaire interactive, filtres, glisser-déposer (selon droits), code couleur par activité.
- **Fiche séance** : Formulaire de création/modification avec recherche intelligente de créneaux, gestion des salles multiples, statut.
- **Gestion des contraintes** : Interface de paramétrage des règles pédagogiques.
- **Mode simulation** : Espace dédié pour tester des scénarios, avec export.
- **Gestion des enseignants** : Liste filtrée, ajout/modification, saisie des indisponibilités récurrentes.
- **Gestion des salles** : Visualisation des occupations par jour, gestion des équipements.
- **Emploi du temps personnel** : Vue simplifiée, export PDF et iCal.
- **Rapports** : Interface de sélection des critères et génération de documents.
